# シンプル電卓プログラム 使用ガイド

## 概要
このプログラムは、コマンドラインで動作するシンプルな電卓です。四則演算（加算、減算、乗算、除算）と履歴表示機能を備えています。

## 使い方
1. ターミナルで `python calculator.py` を実行します。
2. プロンプトが表示されたら、以下の形式でコマンドを入力します。

```
[演算子] [数値1] [数値2]
```

- 例: `add 2 3`（2 + 3 の計算）

### 演算子一覧
- `add` : 加算
- `sub` : 減算
- `mul` : 乗算
- `div` : 除算
- `history` : 計算履歴を表示
- `help` : 使い方を表示
- `exit` : プログラム終了

## 履歴機能
- 直近10件の計算履歴を保存・表示します。

## 注意事項
- 除算で0を指定するとエラーになります。
- 数値以外を入力した場合はエラーとなります。

## サンプル実行
```
> add 5 7
12.0
> div 10 0
Error: Division by zero
> history
--- Calculation History ---
5 + 7 = 12.0
10 / 0 = Error (division by zero)
--------------------------
```
